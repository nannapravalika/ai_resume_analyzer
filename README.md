# AI Resume Analyzer

An AI-powered Resume Analyzer web application developed using Python and Flask that analyzes resumes, extracts technical skills, calculates ATS scores, and predicts suitable job roles.


## Project Overview

This project helps users upload their resumes in PDF format and automatically:

- Extracts resume text
- Detects technical skills
- Calculates ATS compatibility score
- Predicts suitable job role
- Provides quick resume insights

The project demonstrates concepts of:

- Artificial Intelligence
- Natural Language Processing (NLP)
- Data Science
- Web Development

## Installation

### Step 1: Clone Repository


git clone https://github.com/nannapravalika/ai_resume_analyzer.git



### Step 2: Move into Project Folder


cd ai_resume_analyzer


### Step 3: Install Dependencies


pip install -r requirements.txt


## Run the Project


python app.py


Open browser:

http://127.0.0.1:5000


## How It Works

1. User uploads resume PDF
2. System extracts text from resume
3. Skills are matched with predefined skill database
4. ATS score is calculated
5. Suitable job role is predicted
6. Results displayed on web page

## ATS Score Logic

ATS score is calculated based on:

(Number of Matched Skills / Total Skills) × 100
